# Playdough Cutters
![20250223_103542](https://github.com/user-attachments/assets/1fefa936-ce32-470f-8029-b3bc6becea69)
These are simple 3D printed playdough cutters for a butterfly, smiley face, flower and rainbow.
They were modeled in Fusion360 by importing an SVG using its profiles as a template to create the outer cutting section, and a indentation section for details.
