# Small Projects
This repo contains various small projects.

**1. Playdough Cutters:** 3D printed playdough cutters.  
**2. Small Drill Bit Box:** A small box for holding loose drill bits.  
**3. Socket Case:** Case for holding a socket set.  
**4. Connect 4:** Connect 4 board game running in a terminal.  
**5. Electronic Buzzer:** Simple two tone electronic buzzer.  
**6. Differential Screw 1.0:** A simple differential screw that can be 3D printed.  
**7. MATLAB Scripts:** MATLAB scripts written for university assignments that didn't deserve their own folder.  
**8. Analog Conditioning Circuit:** A circuit to condition an analog audio signal from a microphone to the fed into a ADC.
