# Small Projects
This repo contains various small projects.

**1. Playdough Cutters:** 3D printed playdough cutters.  
**2. Small Drill Bit Box:** A small box for holding loose drill bits.  
**3. Socket Case:** Case for holding a socket set.  
**4. Connect 4:** Connect 4 boardgame running in a terminal.  
**5. Electronic Buzzer:** Simple two tone electroic buzzer.
